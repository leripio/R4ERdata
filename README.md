# Installation
You can install the development version from GitHub:

```{r}
install.packages(
"https://github.com/leripio/R4ERdata/releases/download/v1.0.0/R4ER2data_1.0.0.tar.gz",
repos = NULL,
type = "source"
)
```

or

```{r}
install.packages("devtools")
devtools::install_github("leripio/R4ER2data")
```
